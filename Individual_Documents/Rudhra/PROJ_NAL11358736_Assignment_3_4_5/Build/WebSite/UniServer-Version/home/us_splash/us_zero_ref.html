Zero XI Reference page

This reference page is displayed when the Controller and default browser or Controller and portable browser first start. Its purpose, is to provide a timing reference to prevent multiple browser instances opening.

After the above initial condition subsequent browser restarts do not display this reference page.

Reference page is located in folder UniServerZ\home\us_splash\us_zero_ref.html you can change the content of this page.

If you wish to change the page title <title>UniServer Zero 11</title> you must change its corresponding entry in the configuration file C:\UniServerZ\home\us_config\us_config.ini under section [UNIQUE] change parameter Unique_text to match page title.
